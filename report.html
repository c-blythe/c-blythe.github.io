Link: https://c-blythe.github.io/

What the app does:

  • Trivia
  • The app will randomly generate a question
  • When getting a new question, it can be modified by category and difficulty
  • The user is presented with the answers as buttons
  • When an incorrect button is clicked, the speaking entity will say it is incorrect before resaying the question
  • When the correct button is clicked, the speaking entity will say that, and say a new question
  • Cowsay
  • The trivia text is formatted through the Cowsay API, which makes it look like a cow said it
  • The speaking entity, both the one asking the questions and the one responding
  • This is done through the dropdowns, which both default to 'Cow'
  • Trivia
  • Every trivia question that is given is logged in an array
  • The search bar can be used to search through and filter through the array
  • Any word can be searched, and if the question has it, it will be returned
  • Additionally, keywords, preceded by a '!', can be used as well
  • These keywords provide additional filters to the search
  • For example, the search '!solved' will return every solved question

How the app satisfies requirements:
  • The app is deployed publicly (as per the link above)
  • The screen width affects the placement of the buttons and selections
  • The OpenTriviaDB API and Cowsay API are used in conjunction with each other
  • A search for past trivia questions is given
  • These search results are displayed below the searchbar
  • Certain keywords (e.g. 'solved') can be used to filter the results
